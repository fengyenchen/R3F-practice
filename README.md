# React Three Fiber (R3F) 練習

這是一個基於 React Three Fiber (R3F) 的 3D 網頁互動練習。主要用於探索 3D 模型在網頁前端的呈現、材質與燈光效果調校，並結合 Tailwind CSS 進行現代化的介面排版。

---

## 如何在本地端啟動專案

請按照以下步驟，使用 `pnpm` 在你的電腦上執行此專案：

### 1. 安裝依賴

打開終端機進入專案資料夾，執行以下指令安裝所有必備的套件：

```bash
pnpm install
```

### 2. 啟動本地開發伺服器

安裝完成後，執行以下指令啟動開發伺服器：

```bash
pnpm dev
```
